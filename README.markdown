Name
====

riot-openresty

Description
===========

riotjs application using openresty as API

Prerequisites
=============

* openresty
* npm

Installation
============

* `git clone --recursive https://github.com/antonheryanto/riot-openresty`
* `make prepare`
* `make`
* browse [dev](http://localhost:8000) or [dist](http://localhost:8000/dist)
